# AivisSpeech Studio — WebUI

[AIVIS Router](https://github.com/hit1023/me-aivis-router) のブラウザ操作用フロントエンドです。  
Nginx で配信するシンプルな単一HTMLファイル構成です。

---

## 機能

### 🎵 音声合成タブ
- スピーカー・スタイルの選択
- テキスト入力（最大500文字）
- 話速・音高・抑揚・音量・緩急・無音倍率のリアルタイム調整
- MP3再生 & ダウンロード
- VRAMロード状態の確認・強制アンロード
- **モデルインストール** — `.aivmx` ファイルをドラッグ＆ドロップまたはファイル選択でアップロード
- **モデルアンインストール** — インストール済みモデルをワンクリックで削除（確認ダイアログ付き）
- **リアルタイムcurlサンプル** — 現在のパラメータ設定をそのままcurlコマンドで確認・コピー可能

### 📖 単語辞書タブ
- 固有名詞・読み方をユーザー辞書に登録
- **インタラクティブなアクセントビジュアライザー**（スライダーでピッチ曲線をリアルタイムプレビュー）
- ひらがな入力 → 自動カタカナ変換
- 登録単語の検索・編集・削除

### 📝 置換ルールタブ
- 音声合成前に適用するテキスト置換ルールを管理
- 英字固有名詞の誤読対策に有効（例: `Mumon → ミューモン`）
- ルールの追加・検索・削除

> **単語辞書 vs 置換ルール**  
> 英字の単語は辞書に登録するとカタカナに変換されてしまい、
> コンテキストによって効かないケースがあります。
> 英字固有名詞には置換ルールを使うのが確実です。

## 必要な環境

- Docker / Docker Compose
- [AIVIS Router](https://github.com/hit1023/me-aivis-router) が起動していること

---

## セットアップ

```bash
git clone <このリポジトリのURL>
cd AivisWebUI
docker compose up -d
```

ブラウザで `http://localhost:8888` を開きます。

---

## 接続先APIの変更

`.env` ファイルで `AIVIS_API_URL` を設定します。

```env
AIVIS_API_URL=https://your-aivis-router-url
```

コンテナ起動時に `docker-entrypoint.sh` が `index.html` 内のプレースホルダーを自動置換します。
